# Blog System - README

## Project Description

A simple blog system demonstrating web development progression from basic HTML to full-stack development with Node.js, SQLite, and Markdown support.

## Project Structure

```
.
├── v1-basic/          # HTML only version
├── v2-css/           # HTML + CSS version
├── v3-javascript/    # HTML + CSS + JavaScript version
├── v4-nodejs/        # Node.js server version
├── v5-fullstack/     # Full stack with SQLite + Markdown
└── doc/              # Documentation
```

## Quick Start

### Static Versions (v1, v2, v3)
1. Navigate to the version folder
2. Open `index.html` in your browser

### Node.js Versions (v4, v5)

```bash
# For v4-nodejs
cd v4-nodejs
npm install
npm start

# For v5-fullstack
cd v5-fullstack
npm install
npm start

# Open browser to http://localhost:3000
```

## Version Features

| Version | Description |
|---------|-------------|
| v1-basic | Static HTML page with forms |
| v2-css | Styled HTML page |
| v3-javascript | Interactive client-side functionality |
| v4-nodejs | Node.js server with in-memory storage |
| v5-fullstack | SQLite database + Markdown support |

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Node.js
- Express.js
- SQLite3
- Marked (Markdown parser)

## Requirements

- Node.js v14 or higher
- npm (Node Package Manager)
